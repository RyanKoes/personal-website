# Ryan Koes - Personal Portfolio Website

A professional portfolio website showcasing my experience, education, and projects in computer science and engineering.

## Overview

This is my personal portfolio website built with HTML, CSS, and JavaScript. The site features a clean, modern design with interactive elements and animations to provide an engaging user experience.

This website was developed in conjunction with my interactive web design course.

## Features

- **Responsive Design**: Fully responsive layout that adapts to all screen sizes
- **Interactive Animations**: Smooth fade-in effects, skill bar animations, and proximity-based text effects
- **Dynamic Content**: Automated slideshow, scroll-triggered animations, and interactive navigation
- **Professional Presentation**: Comprehensive sections for about, experience, education, and extracurricular activities

## Structure

- `index.html` - Homepage with overview of skills, education, and experience
- `about.html` - Detailed personal background, skills, and photo gallery
- `experience.html` - Professional and research experience timeline
- `education.html` - Academic credentials and achievements
- `extracurricular.html` - Club involvement and personal projects
- `css/` - Stylesheets for each page
- `images/` - Profile photos and project images
- `documents/` - Downloadable resume and transcripts
- `scripts/` - JavaScript files for interactive features
